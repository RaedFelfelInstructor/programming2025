# Introduction to Programming Using C#

A comprehensive 15-week course designed for complete beginners, covering everything from your first line of code to object-oriented programming and modern C# features.

## Course Information

| | |
|---|---|
| **Level** | Level 1 (No prerequisites) |
| **Duration** | 15 weeks |
| **Schedule** | 3 lectures × 45 minutes per week |
| **Language** | C# (.NET) |
| **Tools** | Visual Studio / VS Code, .NET SDK |

## Course Overview

This course is the entry point into a 4-semester program where students ultimately build web applications. It focuses on building a strong programming foundation — from basic syntax and control flow through object-oriented programming — preparing students for later courses in databases, system design, and web application development with MVC.

The course is organized into three progressive blocks:

```
Block 1: Language Fundamentals (Weeks 1–6)
    └── Core syntax, control flow, methods, and collections

Block 2: OOP Foundations (Weeks 7–11)
    └── Classes, inheritance, polymorphism, and interfaces

Block 3: Advanced Language Topics (Weeks 12–15)
    └── Exception handling, generics, LINQ, and integration
```

## What's in Each Week

Every week folder contains six files:

| File | Description |
|------|-------------|
| `README.md` | Week overview, learning objectives, and links to all materials |
| `lecture-1.md` | First lecture — introduces the week's core concepts |
| `lecture-2.md` | Second lecture — deepens understanding with new techniques |
| `lecture-3.md` | Third lecture — advanced patterns and complete examples |
| `exercises.md` | Progressive practice problems from basic to challenging |
| `assignment.md` | Weekly project that integrates all concepts into a practical application |

## How to Use This Repository

**Students:** Navigate to the current week's folder and start with the `README.md` for an overview. Work through the three lectures in order, practice with the exercises, and complete the weekly assignment.

**Instructors:** Each week is self-contained with all materials needed for three 45-minute sessions. Exercises and assignments can be used as-is or adapted to your teaching style. A separate instructor guide with solutions, timing, and teaching narratives is available for each week.

## Tools Setup

To follow along with this course, you'll need:

1. **[Visual Studio 2022 Community](https://visualstudio.microsoft.com/downloads/)** (recommended) or **[VS Code](https://code.visualstudio.com/)** with the C# Dev Kit extension
2. **[.NET SDK](https://dotnet.microsoft.com/download)** (latest LTS version)

Detailed setup instructions are included in [Week 1 — Lecture 1](week%2001/lecture-1.md).
